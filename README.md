# TaskFlow Native (Android)

Native Android companion app for the TaskFlow dissertation project.

## How to run the app

### Prerequisites
- Android Studio (recent stable version)
- Android SDK installed via Android Studio
- At least one emulator image (or a physical Android device)

### Steps
1. Open Android Studio.
2. Select **Open** and choose this folder:
   - `TaskFlow Native (Android) 2365963`
3. Wait for Gradle sync to finish.
4. Start an emulator from **Tools > Device Manager** (or connect a phone with USB debugging enabled).
5. In the top run target, choose the `app` configuration and your device.
6. Click **Run** (green play button).

The app will install and open on the selected device/emulator.

### Optional verification
- Turn off internet on emulator/device to verify offline mode banner/dialog.
- Open **Settings > Account** to test local sign up/sign in/sign out flows.
